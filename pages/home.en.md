# Welcome

This is your personal website. Edit this page by modifying `pages/home.en.md`.

Use the navigation above to explore different sections, or toggle between Arabic and English using the language switcher.

## Getting Started

1. Open `config.yaml` to set your name, bio, social links, and theme color.
2. Replace the content in the `pages/` directory with your own.
3. Add blog posts and updates in `pages/blogs/` and `pages/posts/`.
4. Run `ruby build.rb` after adding or removing entries to regenerate manifests.
